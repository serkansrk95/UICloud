# Usage Flow

## Önerilen Claude çalışma akışı
1. `CLAUDE.md` oku
2. `PROJECT_CONTEXT.md` oku
3. `references/` içindeki görsel niyeti ve markup referanslarını oku
4. `rules/` dosyalarına uy
5. İlgili `skills/` dosyalarını uygula
6. `.claude/commands/implement-anime-detail-redesign.md` görevini çalıştır
7. Son olarak `.claude/commands/review-anime-detail-redesign.md` ile self-review yap
