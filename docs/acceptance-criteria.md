# Acceptance Criteria

## Layout
- [ ] Sol ana akışta eski episodes listesi yok
- [ ] Sağ üstte yeni episodes navigator var
- [ ] Related anime kartı episode navigator'ın altında kalıyor
- [ ] Sayfa genel koyu tema bütünlüğünü koruyor

## Episodes
- [ ] 100'lü blok mantığı var
- [ ] Range seçici var
- [ ] Episode number search var
- [ ] Episode numaraları grid görünümünde
- [ ] Aktif bölüm state'i belirgin
- [ ] Uzun seri senaryosunda tasarım bozulmuyor

## Characters
- [ ] Başlangıçta sadece 2 satır görünüyor
- [ ] Expand ile tüm karakterler açılıyor
- [ ] Collapse ile tekrar kısalıyor

## Synopsis
- [ ] Yazı alanı kontrollü yükseklikte başlıyor
- [ ] Gradient fade veya benzer görsel ipucu var
- [ ] Expand/collapse çalışıyor

## Quality
- [ ] Duplicate content yok
- [ ] Unrelated style drift yok
- [ ] Mobil görünüm bozulmuyor
- [ ] Hover/focus state'ler korunuyor
