# No Regression Rules

1. Comments/Yorumlar bloğu bozulmayacak.
2. Header/hero/meta alanı etkilenmeyecek.
3. Related anime kartı tamamen kaybolmayacak.
4. Episode linkleri çalışmaya devam edecek.
5. Few-item senaryolarında expand butonu gereksizse görünmeyecek.
6. Kısa synopsis varsa gereksiz expand eklenmeyecek.
7. Duplicate episode render bırakılmayacak.
