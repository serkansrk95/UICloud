# Layout Rules

1. Sol ana akış = başlık sonrası içerik, açıklama, karakterler, yorumlar.
2. Sağ sidebar = yeni episodes navigator + related anime.
3. Eski episodes listesi solda kalmayacak.
4. Characters bölümü collapse durumunda 2 satırdan fazla görünmeyecek.
5. Synopsis alanı ilk yüklemede kontrollü yükseklikte olacak.
6. Mobile'da tüm yapı tek kolon akışa düzgün düşecek.
7. Sidebar yapısı dar ekranlarda mantıklı sırayla alta inmeli.
