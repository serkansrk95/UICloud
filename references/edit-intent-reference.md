# Edit Intent Reference

Edit krokisindeki niyet şu şekilde okunmalı:

- Sol tarafta bulunan mevcut episodes alanı kaldırılacak
- Sağ üstte boş bırakılan / işaretlenen alan yeni episode navigator için kullanılacak
- Character alanı daha kısa tutulacak
- Uzun yazı alanı daha kontrollü hale getirilecek
- Ana amaç sayfayı daha kompakt ve daha gezilebilir yapmak
